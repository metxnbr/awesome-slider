# awesome-slider (alpha)

a vanilla javascript slider/carousel/swiper plugin.

[Demo](https://metxnbr.github.io/doc/docs/awesome-slider-demo/)

## usage

### overview

```js
var slider = new Slider(images, container, [options]);
```

### images

数组, 图片的 src

### container

包裹 images 的 dom 元素

### options

```js
options = {};
```
